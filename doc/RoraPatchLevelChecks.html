Overview

This module adds checks for patch level and OS that the database is running on. Currently the patch_level_scan check is at alpha level with a couple of instances enabled. It essentially looks at the md5 of packages which incdicate that a particular CPU has been installed. Ideally this needs updated on a regular basis (quarterly).

116: def detect_os_version
117: #Work out what OS we are
118: #At the moment this is a bit of a hack. The hack is that file names in Windows based OS's start with an Alpha character and in Unix statr with a /
119:   @conn.exec('select FILE_NAME from DBA_DATA_FILES') do |r|
120:     if r[0] =~ /^[a-zA-Z]/
121:       @db_os = 'Windows'
122:     elsif r[0] =~ /^\//
123:       @db_os = 'Unix'
124:     else
125:       @scan_log.WARN("unable to determine OS")
126:       #TODO: move this to a thrown error to be caught elsewhere for now default to unix
127:       @db_os = 'Unix'
128:     end
129:   end
130: end
