Overview

This is the main class for the file based checks that RoraScanner can perform. These are split into checks on the sqlnet and listener files (tnsnames checks TBA) and file permissions checks. The file permissions checks are based on the output of ‘ls -al’ for unix and dumpsec for windows , and basically just review for common permissions errors (eg, use of the everyone group for permissions in windows, and files not belonging to the oracle user in unix.

If you’re looking to add checks to the modules, use the template below for which variables are needed.

Template Variables for Scan Modules

 each module should be named <descriptive>_scan
 and have the following variables set for reporting
 * _source
 * _description
 * _sql
 * _implications
 * _banner
 * _vuln_class
 * _version
 * _column_names
 * _results
